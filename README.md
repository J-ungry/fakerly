1. 이메일 찾기에 성공하면 서버에서 보낸 이메일을 화면에 출력함

2. 로그인하러 가기 버튼 클릭 → 로그인 페이지로 이동

3. 비밀번호 찾기 버튼 클릭 → 이메일/비밀번호 찾기 페이지로 이동

### 비밀번호 찾기 성공 페이지
